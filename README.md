# MechaLapaMobile

Сензорна система, събираща данни от околната среда и предоставяща информация в реално време. Проектът представлява приложение, което работи с модулна система от сензори, подходяща за алпинисти и хора, които обичат да прекарват свободното си време сред природата. Също така включва и социална част където могат да се споделят локации с приятели.


## Инсталация

Клониране на проекта:

```bash
git clone https://github.com/uchkunovdaniel/MechaLapaMobile.git
cd MechaLapaMobile
```

Инсталиране на необходимите библиотеки:

```bash
npm install
```

## Стартиране
```bash
npm run build
npx cap sync
npx cap open ios
```
